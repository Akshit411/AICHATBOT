# AICHATBOT

Sample .NET project for an AI-powered customer support chatbot.

## Features
- Chatbot orchestration
- NLP intent recognition interface
- Knowledge base integration
- Human agent escalation
- Conversation tracking
- Analytics service

## Structure
- `src/AICHATBOT/Models`
- `src/AICHATBOT/Services`
- `src/AICHATBOT/Interfaces`
- `src/AICHATBOT/Enums`
